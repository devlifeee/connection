

# Мессенджер «СВЯЗЬ» — План реализации

## Обзор
Децентрализованный P2P мессенджер для локальной сети. Тёмный интерфейс, три цвета (#0d0d0d, #161616, #4B8EF5), полностью на русском языке. Все данные — моковые, UI полностью интерактивный.

## Дизайн-система
- Тёмная тема: фон #0d0d0d, поверхности #161616, акцент #4B8EF5
- Шрифты: Inter (UI), JetBrains Mono (ID, IP, хеши)
- Скругления 6px (элементы), 10px (модалки)
- Иконки Lucide React повсюду
- Загруженный логотип используется как брендинг приложения

## Страницы и функционал

### 1. Экран загрузки
- Логотип по центру, сменяющийся текст («Сканирование сети...» → «Обнаружение узлов...» → «Установка защищённых каналов...»), синяя полоса прогресса
- Автопереход на регистрацию/главный экран

### 2. Регистрация (одноразовая)
- Логотип, подзаголовок о децентрализации
- Поле имени, автогенерируемый ID (УЗЛ-XXXXXXXX) с кнопкой перегенерации
- 6 геометрических аватаров на выбор
- Кнопка «Войти в сеть»
- Данные сохраняются в localStorage

### 3. Главный экран — трёхколоночный layout
**Левый сайдбар (260px):**
- Профиль с аватаром, именем, ID, статусом (пульсирующая точка)
- Анимированная мини-визуализация сети (4 точки с линиями)
- Навигация: Чаты, Файлы, Звонки, Узлы, Настройки
- Список диалогов с превью, временем, счётчиком непрочитанных
- Каналы (#общий, #команда)

**Центральная панель (гибкая) — переключается по навигации:**

- **Чат:** шапка с инфо собеседника, сообщения (входящие/исходящие с разным фоном), шифрование, системные события, карточки файлов, поле ввода с иконками
- **Файлы:** drag-and-drop зона, активные передачи с прогресс-баром, история в таблице
- **Звонки:** список звонков, модалка входящего звонка, экран активного голосового/видеозвонка с таймером и панелью управления
- **Узлы:** SVG-граф сети (центральный узел + окружающие), сетка карточек узлов с задержкой, статусом и кнопками действий
- **Настройки:** профиль, сеть, приватность, аудио/видео, интерфейс, о приложении — с рабочими тумблерами и полями

**Правая панель (280px):**
- Информация о выбранном узле/собеседнике: аватар, ID, статистика задержки/сообщений/файлов, безопасность, действия

### 4. Системные компоненты
- Toast-уведомления (подключение, файлы, звонки, отключение)
- Пустые состояния с иконками и пульсирующей анимацией
- Модалка входящего звонка с пульсирующим кольцом

### 5. Адаптивность
- На мобильных: одноколоночный layout с навигацией снизу
- Панели переключаются, правая панель — как выдвижной sheet

### 6. Интерактивность (на моковых данных)
- Отправка сообщений (Enter + кнопка), добавление в список
- Переключение разделов навигации
- Drag-and-drop файлов с имитацией прогресса через setTimeout
- Рабочие тумблеры в настройках (состояние в useState)
- Toast-уведомления с автоудалением
- Модалки звонков (принять/отклонить)
- Копирование ID по клику
- SVG-анимация графа сети

### Тестовые данные
4 узла (Алексей, Мария, Иван, Дарья) с диалогом и файлами, как указано в промте.
